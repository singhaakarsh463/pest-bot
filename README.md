# 🌱 Pest Detection Bot

Pest Detection Bot is a Python-based application designed to help identify plant pests and provide useful recommendations such as pesticides and government schemes for farmers.

---

## 🚀 Features

* 🐛 **Pest Detection**
  Analyze input data or images to identify pests affecting crops.

* 🔍 **Search Functionality**
  Find pest-related information quickly.

* 🌿 **Pesticide Recommendations**
  Suggest suitable pesticides for detected pests.

* 🏛️ **Government Schemes**
  Provide relevant subsidy and scheme information.

* 📊 **Graph-Based Analysis**
  Visualize pest-related data.

---

## 🛠️ Tech Stack

* Python
* JSON (data handling)
* Basic data analysis
